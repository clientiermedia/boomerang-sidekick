# Boomerang Sidekick

A clean, production-ready AI chat interface built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the chat interface.

## Features

- Full-page chat interface with fixed header and input area
- Scrollable message history
- User messages (right-aligned, purple background)
- Assistant messages (left-aligned, white background)
- Auto-scroll to bottom on new messages
- Enter to send, Shift+Enter for new line
- Responsive design

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Hooks (useState only)
